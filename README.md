# 認識星空互動教材

這是一個可直接用瀏覽器開啟的靜態互動教材，適合國小高年級到國中自然課使用。

## 教學重點

- 認識春、夏、秋、冬常見代表星座。
- 理解星等數字越小代表越亮。
- 觀察光害對暗星可見度的影響。
- 練習用方位、季節、亮星、星色與星座形狀尋找夜空線索。
- 理解星空會因地球自轉而繞北天極附近旋轉。
- 理解低空星光會因大氣消光與擾動而變暗、閃爍。
- 閱讀 10 則星星故事，連結文化傳說與實際觀星線索。

## 使用方式

直接開啟 `index.html`，或在本資料夾啟動本機伺服器：

```powershell
python -m http.server 7791
```

再到 `http://localhost:7791/`。

## 課堂建議

1. 先讓學生在「星圖」頁選擇季節，描述最容易辨認的星座形狀。
2. 調整「觀察時間」或勾選播放動畫，觀察星座如何繞北極星附近旋轉。
3. 調整「光害程度」，請學生預測哪些星會先消失，再操作驗證。
4. 切到「概念」頁說明星軌、星星閃爍、大氣消光，以及星座線的意義。
5. 切到「故事」頁，讓學生選一則故事並回到星圖找對應星座或亮星。
6. 用「測驗」頁做 10 題形成性評量。

## 第二階段真實性修正

- 主星圖不再用水平平移表現時間，而是以北方天空常見的「繞北天極附近旋轉」表現一晚中的視運動。
- 星星接近地平線時會套用大氣消光，低空星會變暗，部分亮星也會有較明顯閃爍。
- 光害會提高天空背景亮度，讓暗星、星團與模糊天體較早消失。
- 亮星加入基本星色提示，例如大角星偏橙、參宿四偏紅、織女星與天狼星偏藍白。
- 概念頁加入星軌、大氣消光與閃爍示意動畫。
- 故事頁加入 10 則星星與星座故事，每則都附上對應的科學觀察線索。

## 模型限制

星圖是更接近真實觀星經驗的概念模型，用來呈現季節星座、相對亮度、光害、大氣消光與視運動；它不是精密天文軟體，不能用來查詢特定日期、時間、地點的真實星空位置。
